# Miruro.tv Discovered API Endpoints

## Pipe API (Behind Cloudflare WAF)
- GET /api/secure/pipe?e=<base64url> - Core pipe API (all operations)
  - path=config - Provider configuration
  - path=info/{anilistId} - Anime details by AniList ID
  - path=episodes?anilistId={id} - Episode list
  - path=sources?episodeId={id}&provider={key}&category={type} - Video sources
  - path=search/browse - Browse/search
  - path=search - Search
  - path=schedule - Airing schedule
  - path=reports - Reports/submissions
- POST /api/secure/pipe (JWE ECDH-ES + A256GCM) - Same operations with JWE encryption

## Public Endpoints (No Cloudflare)
- GET /api/events - SSE event stream (version, config, JWKS rotations)
- GET /health - Health check
- GET /random-pool.json - AniList ID list

## Protected Endpoints (Behind Cloudflare)
- GET /api/secure/jwks - JWK set for JWE encryption
- GET /api/mal/me - MyAnimeList user info (requires OAuth token)

## OAuth
- GET /callback - OAuth callback handler (AniList & MyAnimeList)

## Static Assets
- GET /env2.js - Client-side secrets (publicly cached)
- GET /sw.js - Service worker
- GET /registerSW.js - SW registration
- GET /assets/*.js - Rollup/Vite bundles (50+ files)
- GET /assets/*.css - CSS bundles
- GET /manifest.webmanifest - PWA manifest

## Status/Infrastructure
- GET https://status.miruro.com/api/status-page/miruro - Mirror status
- GET https://status.miruro.com/api/status-page/heartbeat/miruro - Mirror heartbeats

## Third-Party
- GET https://graphql.anilist.co - AniList GraphQL API
- GET https://llvpn.com/tag.min.js - Monitag ad script
- GET https://vault01.ultracloud.cc/ - Proxy server A
- GET https://vault02.ultracloud.cc/ - Proxy server B
